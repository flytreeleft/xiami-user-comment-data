## 简介

<div>
<p><br>
</p><p style="border: 0px; padding-top: 26px; color: rgb(0, 0, 0); font-family: 宋体, simsun, sans-serif, Arial; line-height: 26px;"><span style="color: rgb(255, 0, 0);">在爱情面前，人人都是平等的，都有追求幸福的权利。女孩子也应该放下矜持，大胆的去追寻属于自己的爱情。在张静的新歌《痛痛快快爱一场》中，为身边的女孩子们加油打气，谁说女生就只能被动？痛痛快快的爱一次，让爱情在酣畅淋漓中迸发出新的活力。</span></p><br>
<p style="border: 0px; padding-top: 26px; color: rgb(0, 0, 0); font-family: 宋体, simsun, sans-serif, Arial; line-height: 26px;"><span style="color: rgb(255, 0, 0);"> 听张静的歌，从来都不会听到那些玻璃心、小矫情、公主病。张静的活力和直爽，是通过每个字节、每个音符都感染到你的。在这首《痛痛快快爱一场》中，张静为女生开启了一扇追求幸福的新大门：既然人生道路漫长，前途不可预测，为什么不能放手去好好爱一场呢？不想要贪图富贵，也不想爱慕虚荣，在爱情面前人人都是平等的，都是自由的。就让年轻的人一起来谈场恋爱吧，不去管那些坚硬的现实，而是遵循内心的呼唤，跟着爱的感觉走，收获幸福的节奏。</span></p><br>
<p style="border: 0px; padding-top: 26px; color: rgb(0, 0, 0); font-family: 宋体, simsun, sans-serif, Arial; line-height: 26px;"><span style="color: rgb(255, 0, 0);"> 在这首新歌中，张静大胆的融入了民歌曲风。在她看来，民歌直率、真性情，那些对于情爱的话，不必要遮遮掩掩的藏在隐晦的歌词中，既然真的想去爱，为什么不大大方方的表达出来呢？所以从张静的新专辑中，能听到她对爱的呼唤，对爱的直接。这份热情和率真，也成为了歌曲最具有感染力的部分。张静希望用歌声去唤醒人们对爱情的向往和追求，让感情不被青春辜负。</span></p><br>
