## 简介

<div>
<span style="font-size:9pt">冰岛到底有多少暧暧内含光的寂寞忧郁灵魂，光是一个</span><span style="font-size:9pt">Sigur Ros</span><span style="font-size:9pt">就可以让我们拨不开那冰凉的朝雾。但是这个地方特产的冷冽质感都会从他们的作品微微渗出，让人感觉寒冷的疏离，亦或是微温的感动。这个看似神秘的岛屿给了我们许多惊喜，也增添了它冰雾垄罩的美好气质。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt">Bardi Johannsson</span><span style="font-size:9pt">是个默默无名的才子，他也是冰岛乐团</span><span style="font-size:9pt">”Bang Gang”</span><span style="font-size:9pt">的主脑，而乐团的主要成员</span><span style="font-size:9pt">也只有他一个。你或许会想：这种化名乐团的一人</span><span style="font-size:9pt">Solo</span><span style="font-size:9pt">不是已经在</span><span style="font-size:9pt">Indie-Eletronica</span><span style="font-size:9pt">界是老套的招数了吗</span><span style="font-size:9pt">?</span><span style="font-size:9pt">但是</span><span style="font-size:9pt">Bang Gang</span><span style="font-size:9pt">的音乐绝对不老套，甚至多变性以及丰富性高的惊人。如果你是喜欢</span><span style="font-size:9pt">Indie-Pop</span><span style="font-size:9pt">的乐迷，那绝对不能遗忘了这张远在冰岛的聪慧之作。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　不同于其它电音同辈，</span><span style="font-size:9pt">Bardi Johannsson</span><span style="font-size:9pt">受到英国</span><span style="font-size:9pt">Trip-Hop</span><span style="font-size:9pt">风潮影响很深，但是他取集了</span><span style="font-size:9pt">Trip-Hop</span><span style="font-size:9pt">流行的部份，把嘻哈的部份褪去，加进了许多流畅的元素，甚至包括了</span><span style="font-size:9pt">Brit-Pop</span><span style="font-size:9pt">。这种原型的音乐概念在</span><span style="font-size:9pt">Bang Gang</span><span style="font-size:9pt">的第一张专辑</span><span style="font-size:9pt">”You”</span><span style="font-size:9pt">可以说是俯拾即是，而他更利用了</span><span style="font-size:9pt">Trip-Hop</span><span style="font-size:9pt">界的特色，那就是专门请甜美清澈女生来帮自己发声，加上他自己纤细透明的嗓音，交织了层次分明的声音质感，也为自己谱写的优美节奏找到最棒的合作伙伴。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　这次的焦点是他们的第二张专辑</span><span style="font-size:9pt">”Something Wrong”</span><span style="font-size:9pt">，这次</span><span style="font-size:9pt">Bardi Johannsson</span><span style="font-size:9pt">放弃大量取样的声效以及</span><span style="font-size:9pt">Loop</span><span style="font-size:9pt">的电子节奏，反而是回归民谣清新般的创作，钢弦民谣吉他轻拨刷动几乎在每一首歌里面跳动，一反过去冰冷凄美的姿态，又辟出另一个温暖甜美的草原。不过这也没有让</span><span style="font-size:9pt">”Something Wrong”</span><span style="font-size:9pt">变为一张纯粹的一把吉他专辑，电子节奏、编曲还是在骨干下不断地进行巨大的影响，造就了这张专辑多变的层次性和丰富的音乐性，一张优美的</span><span style="font-size:9pt">Indie-Pop</span><span style="font-size:9pt">专辑焉然诞生。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt">Esther Talia Casey</span><span style="font-size:9pt">、</span><span style="font-size:9pt">Phoebe Tolmer</span><span style="font-size:9pt">、</span><span style="font-size:9pt">Keren Ann Zeidel</span><span style="font-size:9pt">、</span><span style="font-size:9pt">Daniel Agust Haraldsson</span><span style="font-size:9pt">和</span><span style="font-size:9pt">Nicolette Suwoton</span><span style="font-size:9pt">这些主音客席一字排开，有冰岛美女、法国女生，乐团的主唱，这些人不只有献声而已，还参与了创作和编曲，所以可以想象专辑的乐风应该是兼容并蓄，并且由</span><span style="font-size:9pt">Bardi Johannsson</span><span style="font-size:9pt">统合出相同的概念，那就是一场美丽的邂遘，或是充满诗意的错误。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt">“Inside”</span><span style="font-size:9pt">柔顺的弦乐和优美旋律，宛如直入梦境般的甜腻，但是后面却跳出了</span><span style="font-size:9pt">Trip-Hop</span><span style="font-size:9pt">的节奏，似乎这才是它的出生地，轻盈地飘浮在空气中。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　接着的</span><span style="font-size:9pt">”Something Wrong”</span><span style="font-size:9pt">更是彷佛</span><span style="font-size:9pt">Massive Attack</span><span style="font-size:9pt">上身一样，从背后深层黑暗的合成器，到</span><span style="font-size:9pt">Esther Talia Casey</span><span style="font-size:9pt">晶莹剔透的声线，活脱脱就是从那个年代穿梭的焦躁作品。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　而好听到不能的</span><span style="font-size:9pt">”It’s Alright”</span><span style="font-size:9pt">反而呈现了清新的民谣流行路线，那种刷动的美妙质感直逼大西洋对岸的</span><span style="font-size:9pt">Death Cab For Cutie</span><span style="font-size:9pt">，但是组合的和弦却是不折不扣的英摇痛，十分有趣。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt">“Forward And Reverse”</span><span style="font-size:9pt">马上令人回想起</span><span style="font-size:9pt">Slowdive</span><span style="font-size:9pt">的抒情之作，略带迷幻也更显慵懒，扭曲的吉他声响彷佛是</span><span style="font-size:9pt">Keren Ann Zeidel</span><span style="font-size:9pt">声线的最好良伴，后面更是以爆炸的声响作收。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　而</span><span style="font-size:9pt">”Find What You Get”</span><span style="font-size:9pt">马上来个大地震，是一首不折不扣的摇滚歌曲，感受不到任何电离子，节奏似乎是走着</span><span style="font-size:9pt">Post-Rock Revival</span><span style="font-size:9pt">的调调，但是却是不折不扣的</span><span style="font-size:9pt">Brit-Pop</span><span style="font-size:9pt">旋律，间奏及结尾的激烈乐器演译令人目瞪口呆，大呼过瘾。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　接着而来的</span><span style="font-size:9pt">”In the Morning”</span><span style="font-size:9pt">却有着扭曲的草根蓝调开头，不过中段却转折为沧桑的苦情民谣，</span><span style="font-size:9pt">Daniel Agust Haraldsson</span><span style="font-size:9pt">化身为西部的漂流过客，唱着忧郁寂寞的故事。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt">“Stop In the Name Of Love”</span><span style="font-size:9pt">这首番石榴大作到了</span><span style="font-size:9pt">Bardi Johannsson</span><span style="font-size:9pt">的手中马上变成清新如</span><span style="font-size:9pt">Belle &amp; Sebastian</span><span style="font-size:9pt">的梦境民谣，宛如漫步在乡间小路上，随口轻松哼出的可人歌曲。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　而</span><span style="font-size:9pt">”Contradiction”</span><span style="font-size:9pt">则让</span><span style="font-size:9pt">Nicolette Suwoton</span><span style="font-size:9pt">歌声作为主角，呈现出浓浓地复古风情，犹如时光倒错的回到了</span><span style="font-size:9pt">40</span><span style="font-size:9pt">年代，但却流露出阵阵的哀伤。最后静谧演奏歌曲说明了专辑多样的美好都要回归于</span><span style="font-size:9pt">Indie-Pop</span><span style="font-size:9pt">的优美质感里，一切美好尽在不言中。</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　</span><span style="font-size:9pt"> </span><br><br>
<span style="font-size:9pt">　　我们是在敦南诚音的试听架发现它的，一直挣扎到高雄的大统诚音才以最低价购入，细听之下真的惊为天人，果然相信自己的耳朵是对的，这张专辑提供了许多优美流畅的元素，让人爱不释手。更令人惊喜的是，这张专辑也来自恐怖的</span><span style="font-size:9pt">2003</span><span style="font-size:9pt">年，却几乎没有人提起它。如果你</span><span style="font-size:9pt">”</span><span style="font-size:9pt">不幸</span><span style="font-size:9pt">”</span><span style="font-size:9pt">在诚音发现它，价钱一定是非常夸张的，但是起务必把它带回家，因为它能满足你对</span><span style="font-size:9pt">Indie-Pop</span><span style="font-size:9pt">的所有想象，喔，甚至超乎你的想象。</span>
</div>
