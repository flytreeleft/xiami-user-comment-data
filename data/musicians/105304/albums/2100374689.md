## 简介

<div>
<p><br>
莲，被誉为纯洁和圣洁的象征。莲开在现实世界而又时常开在你我的梦中，睡觉与醒来----亦是觉与醒。<br><br>
在这个喧嚣的世界里最需要的或许就是这觉醒的力量。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<br><br>
“莲，你静静绽放在水里，花，你随风吹起在空中。” &nbsp; 如梦如幻的心灵世界，就是那样的静谧而美好。<br><br>
睡莲---让我们的世界安静下来，让我们的心灵慢慢的觉醒。让觉醒带着爱的力量，让觉醒带来这更加和平的世界。<br>
2016，盛夏时节，让我们又见善音歌者桑吉平措，一起走进《睡莲》，走近心灵的梦幻世界。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<br><br>
让我们的世界安静下来，“莲，你静静绽放在水里”---你就是如此的美丽。。。<br>
</p>
</div>
